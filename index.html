install-Latex

Published on: Apr 3rd, 2013
Tags:

1、下载在线安装LaTex脚本：

install-tl-unx.tar.gz 再按如下命令安装:

$tar -xvf install-tl-unx.tar.gz
$cd install-tl-20130402
$sudo ./install-tl
[... messages omitted ...]
Enter command: i
[... when done, see below for post-install ...]

tips:
如果以前装过旧版本的Tex,则按如下命令清除：

$sudo rm -rf /usr/local/texlive/2012
$rm -rf ~/.texlive2012

当然上面安装的方法为text模型安装，下面则为GUI图形界面安装过程：

$tar -xvf install-tl-unx.tar.gz
$cd install-tl-20130402
$sudo apt-get install perl-tk
$sudo ./install-tl --gui=perltk

接下来就是看图说话咯
so easy!
漫长的等待安装结束后基本就算安装成功了，接下来就是环境变量的设置了
设置方法按如下：

$sudo gvim /etc/profile

然后在profile中添加如下语句:

export PATH="$PATH:/usr/local/texlive/2012/bin/i386-linux"
export MANPATH="$MANPATH:/usr/local/texlive/2012/texmf/doc/man"
export INFOPATH="$INFOPATH:/usr/local/texlive/2012/texmf/doc/info"

测试是否安装成功如下：

$tex --version
TeX 3.1415926 (TeX Live 2012)
kpathsea version 6.1.0

There is NO warranty.  Redistribution of this software is

named COPYING and the TeX source.

使整个系统中能使用TEXLive的字体：

$sudo cp /usr/local/texlive/2012/texmf-var/fonts/conf/texlive-fontconfig.conf /etc/fonts/conf.d/09-texlive.conf
$sudo fc-cache -fsv

学习markdown

Published on: Apr 2nd, 2013
Tags:

1、段落，标题，区块引用

I、Markdown段落就直接写，不能有制表符， 段落之间的空行可以直接按回车键空行即可但不能tab键再回车。
换行在上一行的结尾添加两个以上的空格
paragraph example:

Dear ZLove:
This is the first paragraph.

This is the last paragraph .

Result:
Dear ZLove:
This is the first paragraph.

This is the last paragraph .
II、Markdown标题支持两种标题的语法Setext与axt
a)、Setext利用底线形式
’=’顶级标题H1，’-‘二级标题H2
b)、axt利用在行首如’#’;
’#’的数量从一到六，表示H1~H6级标题

Title example:

This is one H1 title
==================
This is one H2 title
------------------
#This is another H1 title
##This is another H2 title
........
######This is one H6 title

reslut:

This is one H1 title

This is one H2 title

This is another H1 title

This is another H2 title

……..

This is one H6 title

III、Markdown区块引用应用email的’>’尖括号。
tips:
区块引用可以嵌套（例如：引用内的引用），只要根据层次加上不同数量的 ‘>’
引用的区块内可以使用其他的 Markdown 语法，包括标题、列表、代码区块等
example:

>This is a blockquote.
>This is the second paragraph in the blockquote.
>## This is an H2 in a blockquote

Result:

This is a blockquote.
This is the second paragraph in the blockquote.

This is an H2 in a blockquote

2、列表、区块代码、分隔线

I、Markdown列表支持有序和无序列表。
a)、无序列表在行首使用’+’、’-‘、’*’作为列表标记.
b)、有序列表在行首使用阿拉伯数字1,2,3加.作为列表标记.
tips:
有序列表中的数字标记不影响输出的HTML结果
列表项目可以包含多个段落，每个项目下的段落都必须缩进 4 个空格或是 1 个制表符
example:

* red
* bule
* gree
5. red
2. bule
3. green

result一样:
* red
* bule
* green
tips、如果你想写下面的语句，则需用’\转移

2008. This is a beautiful year
则需写成2008\. This is a beautiful year

2008. This is a beautiful year

II、区块代码
Markdown 中建立代码区块很简单，只要简单地留出一空行并在行首缩进 4 个空格或是 1 个制表符就可以，代码区块里面’&’ 、 ‘<’ 和 ‘>’ 会自动转成 HTML 实体,不用理会
Markdown区块代码语法如下

This is regular paragraph

    <code>

example:

This a regular paragraph
#include<stdio.h>
int main()
{
  char name[30];
  int n;
  scanf("%s %d",name,&n);
  printf("%s",name);
return 0;
}

result:
This a regular paragraph

#include<stdio.h>
int main()
{
  char name[30];
  int n;
  scanf("%s %d",name,&n);
  printf("%s",name);
return 0;
}

III、分隔线
Markdown分隔线语法：
在一行中用三个以上的星号、减号、底线来建立一个分隔线，行内不能有其他东西。你也可以在星号或是减号中间插入空格
example:

* * * *
---
-----------------
_____________

result:

3、连接、强调、代码

I、连接包含两种语法：
a)、行内式

[link](http://....com "Title")

b)、参考式

[link][id]
[id]: http://....com "Title"

tips:
Title为可选参数;
链接内容规定如下：
id可以是字母、数字、空白和标点符号，但是并不区分大小写
title 属性放到下一行，也可以加一些缩进
方括号（前面可以选择性地加上至多三个空格来缩进），里面输入链接文字
接着一个冒号
接着一个以上的空格或制表符
接着链接的网址
选择性地接着 title 内容，可以用单引号、双引号或是括弧包着

example:

This is zulove's [home page](http://zulove.github.com "ZLove").
This is zulove's farovite [sites][hack].
[hack]: https://news.ycombinator.com/ 'ZLove'

result:
This is zulove’s home page.

This is zulove’s farovite sites.

tips:

隐式链接标记功能让你可以省略指定链接标记，这种情形下，链接标记会视为等同于链接文字，要用隐式链接标记只要在链接文字后面加上一个空的方括号
链接到同样主机的资源，可以使用相对路径:

[Google][]
[Google]: http://google.com/
See my [About](/about/) page for details.

Google

II、强调

Markdown使用()和下划线(_)表示强调：
一个星号()或一个(_)下划线被认为是[em][/em]
两个星号(**)或连个下划线(__)被认为是[strong][/strong]
example:

This is a *human*
This is a big __humman__

result:
This is a human
This is a big humman tips:
如果 * 和 _ 两边都有空白的话，它们就只会被当成普通的符号
如果要在文字前后直接插入普通的星号或底线，你可以用反斜线

\*this is beautiful\*

*this is beautiful*
III、代码
Markdown标记一小段行内代码用(‘)
如果要在代码区段内插入反引号，你可以用多个反引号来开启和结束代码区段.
代码区段的起始和结束端都可以放入一个空白，起始端后面一个，结束端前面一个，这样你就可以在区段的一开始就插入反引号

这个是C打印函数 'printf()' ;

这个是C打印函数 ‘printf()’ ;

3)、图片、自动链接、反斜杠

I、Markdown插入图片有两种方式：行内式和参考式

![Alt text](/path/to/img.jpg)
![Alt text](/path/to/img.jpg "title")
![Alt text][id]
[id]: url/to/image  "title"

tips:
一个惊叹号 !
接着一个方括号，里面放上图片的替代文字
接着一个普通括号，里面放上图片的网址，最后还可以用引号包住并加上 选择性的 ‘title’ 文字。
[id]是图片参考的名称，图片参考的定义方式则和连结参考一样
Markdown 没有办法指定图片的宽高，如果你需要的话，你可以使用普通的 标签
II、自动连接

<http://>

http://zulove.github.com
III、反斜杠
Markdown 可以利用反斜杠来插入一些在语法中有其它意义的符号
Markdown 支持以下这些符号前面加上反斜杠来帮助插入普通的符号:

\   反斜线
`   反引号
*   星号
_   底线
{}  花括号
[]  方括号
()  括弧
#   井字号
+   加号
-   减号
.   英文句点
!   惊叹号
